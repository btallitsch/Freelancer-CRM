# Folio CRM

A lightweight CRM for freelancers and solo operators. Built with Next.js, React, and TypeScript.

## Features

- **Dashboard** — revenue, pipeline value, active clients, avg deal size
- **Clients** — card grid with full detail view, tags, status tracking
- **Pipeline** — drag-and-drop Kanban across 5 stages
- **Notes** — activity log tied to clients

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

```bash
git init && git add . && git commit -m "initial commit"
# push to GitHub, then import at vercel.com/new
```

No environment variables or database required.
